The CORS error indicates that your backend API is not correctly configured to handle requests with credentials from different origins. When using withCredentials: true in Axios, you cannot use the wildcard * for the Access-Control-Allow-Origin header. Instead, you need to specify the exact origin that is allowed to make the request.

Update your server CORS configuration in index.js to handle this correctly.

// Define allowed origins
const allowedOrigins = [process.env.REACT_URL] ;

// CORS configuration
app.use(cors({
origin: function (origin, callback) {
// Allow requests with no origin (like mobile apps or curl requests)
if (!origin) return callback(null, true)                                                      ;
if (allowedOrigins.indexOf(origin) === -1) {
const msg = 'The CORS policy for this site does not allow access from the specified origin.';
return callback(new Error(msg), false)                                                        ;
}
return callback(null, true)                                                                   ;
},
credentials: true // Allow credentials (cookies, authorization headers, etc.)
}))                                                                                           ;
